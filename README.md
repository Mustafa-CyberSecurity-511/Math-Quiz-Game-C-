# Math Quiz Game with C++
A simple math quiz game written in C++ with multiple difficulty levels and operations.

## Features
- Choose number of questions (1 to 10)
- Select difficulty level: Easy, Medium, Hard, or Mix
- Select operation type: Addition, Subtraction, Multiplication, Division, or Mix
- Shows correct and wrong answers
- Final result: Pass or Fail
- Play again option

## How to Run
1. Compile the code:
```bash
g++ main.cpp -o MathQuiz
./main
